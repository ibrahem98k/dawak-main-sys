# Pharmacy-UI

A modern pharmacy management system built with React, Vite, and Tailwind CSS.

## Features

- **Pharmacy Dashboard**: Manage orders, browse suppliers, and track inventory.
- **Supplier Dashboard**: Manage medicines, track orders, and view analytics.
- **Integrated Maps**: Browse pharmacy and supplier locations.
- **Order Validation**: Stock-aware validation for pharmacy orders.

## Tech Stack

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Components**: Radix UI / Shadcn
- **State Management**: TanStack Query
- **Routing**: Wouter

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
